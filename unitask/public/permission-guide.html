Notification Permission Guide

You've previously blocked notification permissions for UniTask. This guide will help you reset those permissions so you can receive important message notifications.

Google Chrome

  1. Click the lock icon (or info icon) in the address bar.
  2. Click Site settings.
  3. Under "Permissions", find Notifications and change it to Allow.
  4. Refresh the UniTask page.

Mozilla Firefox

  1. Click the lock icon in the address bar.
  2. Click Connection secure or More Information.
  3. Click Permissions tab.
  4. Find Notifications and uncheck Block, then select Allow.
  5. Refresh the UniTask page.

Safari

  1. Click Safari in the menu bar.
  2. Select Settings or Preferences.
  3. Go to the Websites tab.
  4. Click on Notifications in the sidebar.
  5. Find UniTask in the list and select Allow.
  6. Refresh the UniTask page.

Microsoft Edge

  1. Click the lock icon in the address bar.
  2. Click Site permissions.
  3. Find Notifications and change to Allow.
  4. Refresh the UniTask page.
